mzTab file format

The mzTab data exchange format is used for communicating mass-spectrometry-based proteomics and metabolomics experimental results. Griss J et al. The mzTab data exchange format: communicating mass-spectrometry-based proteomics and metabolomics experimental results to a wider audience. Mol Cell Proteomics. 2014 Oct;13(10):2765-75. doi: 10.1074/mcp.O113.036681. Epub 2014 Jun 30. PMID: 24980485; PMCID: PMC4189001.

The mzTab format is a tab-delimited text file consisting of a metadata section, a protein section, a peptide section which is not recommended in "Identification" files, and a PSM section as well as a small molecule section. Crux supports the mzTab format for both "Quantification" and "Identification" files. Crux supports the version 1 of the mzTab format.
For a detailed explanation of the mzTab format, please see the mzTab format specification

Meta Data Section

Name Description
mzTab-version It specificaties the mzTab version being used.
mzTab-mode It specifies the mode of the experiment which was conducted (either "Summary" or "Complete").
mzTab-type It specifies the type of experiment which was conducted (either "Quantification" or "Identification").
description It contains a short description of the experiment that was conducted.
